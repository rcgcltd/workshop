Boxes, boxes, it's all about boxes

Most of the HTML elements on your page can be thought of as boxes sitting on top of each other.

  • padding, the space just around the content (e.g., around paragraph text)
  • border, the solid line that sits just outside the padding
  • margin, the space around the outside of the element

Other css properties..

  • width (of an element)
  • background-color, the color behind an element's content and padding
  • color, the color of an element's content (usually text)
  • text-shadow: sets a drop shadow on the text inside an element
  • display: sets the display mode of an element (don't worry about this yet)

Now add the following to your "style.css"

Change the page color..


                            html {
                              background-color: #00539F;
                            }

Sort the body out..


                          body {
                              width: 600px;
                              margin: 0 auto;
                              background-color: #FF9500;
                              padding: 0 20px 20px 20px;
                              border: 5px solid black;
                            }

Continue styling..

Style the main title


                            h1 {
                              margin: 0;
                              padding: 20px 0;
                              color: #00539F;
                              text-shadow: 3px 3px 1px black;
                            }

Center the image..


                            img {
                              display: block;
                              margin: 0 auto;
                            }

Feel free to play around with the styles and css properties

Supercharging our website, with:

javascript

create a file named "main.js"

include in your "index.html" as follows, just before the end of body tag:

Get another image, name it "logo2.png" in addition to the first image "logo1.png" in a folder images

Now open "main.js" and put the following:


                        var myImage = document.querySelector('img');

                        myImage.onclick = function() {
                            var mySrc = myImage.getAttribute('src');
                            if(mySrc === 'images/logo1.png') {
                              myImage.setAttribute ('src','images/logo2.png');
                            } else {
                              myImage.setAttribute ('src','images/logo1.png');
                            }
                        }
